Getting Started

1

Prerequisites

Before accessing the application, ensure you have the following installed:

  • Node.js
  • npm or pnpm
  • Java Coding Pack for VSCode
  • Visual Studio Code

Optional

If you can run Java in another IDE while using VSCode for the frontend, you may skip this, but VSCode is preferred for consistency.

2

Cloning the Repository

Once you have installed the prerequisites, clone this repository to your machine:

Shell
git clone https://github.com/Morfusee/MO-IT110-OOP.git

For more details on cloning a repository, check GitHub's documentation.

3

Running the Application

Frontend

  1. Open the project folder in Visual Studio Code.
  2. Open the terminal.
    1. To open it you can either:
      1. Press Ctrl + J to open the terminal.
      2. Navigate over to Terminal and select New Terminal.
  1. Using the terminal, navigate to the frontend directory:
Shell
cd frontend
  1. Install the packages for the frontend:
npm
Shell
npm install
pnpm
Shell
pnpm install
  1. After installing the packages, start the frontend server:
npm
Shell
npm run dev
pnpm
Shell
pnpm run dev
  1. The frontend server will start and provide a URL where you can access the application.

Backend

Optional

If you prefer another Java-supported IDE, you can run the backend from there and then access the frontend normally.

  1. Open VSCode. This will then initialize the Java SDK:
  1. Wait until the Java SDK is fully loaded.
  1. Click on any Java file, and a Run button should appear beside your open files.
  1. Click the Run button, and a dropdown will appear.
  1. Select MotorphApplication.java.
  2. Once started, the console will display logs indicating that MotorphApplication has started successfully.
