System Actors

Actor

Description

HR Manager

Manages employees, attendance, payroll, leave approvals

Employee

Views attendance, requests leave, checks payroll

Authentication Service

This is just basically a part of the backend that is acting on the log in action of the Employee and HR Manager.

1

Users

Column Name

Data Type

Description

id

INT

Unique identifier for the user account.

username

VARCHAR(100)

Username for logging into the system.

password

CHAR(60)

Hashed password for the user account.

Data Type

Description
