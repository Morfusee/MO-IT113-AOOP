Using the Payroll System

Logging In

  1. Head over to http://localhost:5173/.
  2. Enter your username and password on the login page.
  3. Click the Login button.
  4. Upon successful login, you will be redirected to the Attendance or Employee page.

Note

Initially, the HR Manager's credentials will be provided automatically. You can also log in as an employee.

Attendance Management

1

View Attendance

On the Attendance page, both HR Managers and Employees can view attendance records and access attendance analytics.

HR Manager

Viewing own attendance

  1. Navigate to the Attendance page.
  2. Select the month and year from the dropdown.
  1. If attendance data exists for the selected period, it will be displayed in a table.

Viewing another employee’s attendance

  1. Navigate to the Attendance page.
  2. Use the employee selection combobox beside the month and year dropdown to choose an employee.
  1. Select the month and year from the dropdown.
  2. The selected employee’s attendance data will be displayed in a table.
Employees

Viewing Own Attendance

  1. Navigate to the Attendance page.
  2. Select the month and year from the dropdown.
  1. If attendance data exists for the selected period, it will be displayed in a table.


Employee Management

Important: This feature set is only accessible by the HR Manager.

1

View Employee List

To view the employees of MotorPH, the HR Manager can simply head over to the Employee page.

2

Search Employees

The HR Manager can also search for an employee using the search bar at the top of the page.

Note

If there are no employees found, the table will just return an error message.

3

Create Employees

To create an employee on this page, the HR Manager must:

  1. Click the Create button placed on the top right of the table.
  1. Fill out all the necessary fields.
  2. Finally, click the Create Employee button.
  3. If successful, the created employee should appear at the bottom of the list.

Note

Passwords are auto-generated in the backend. Learn more here.

4

Delete Employees

  1. Click the three-dot icon at the end of the row for the employee you want to delete.
  2. In the dropdown menu, click Delete.
  1. Confirm the action by clicking Delete in the modal prompt.
5

Edit Employees

To edit an employee on this page, the HR Manager must:

  1. Similar to deleting an employee, click the three-dot icon at the end of the row for the employee you want to edit.
  2. In the dropdown menu, click Edit.
  3. In the Edit Employee modal, update the necessary details.
  1. Click the Edit Employee button once done.

Warning

Leaving fields empty will prevent the employee from being edited.


Payroll Management

1

Calculate Monthly Salary

Employees, including the HR Manager, can calculate for their monthly net salary.

HR Manager

Calculating own salary

  1. Select the desired year from the dropdown.

Tip

If the selected year has no payroll months to display, try using 2022.

  1. Select the desired payroll month from the list.
  2. The payroll breakdown for the selected month will be displayed in the right corner of the screen.

Calculating other employee's salary

  1. Select the desired year from the dropdown.
  2. Use the employee selection combobox beside the month and year dropdown to choose an employee.
  3. Select the desired payroll month from the list.
  1. The payroll breakdown for the selected month will be displayed in the right corner of the screen.
Employees

Calculating own salary

  1. Select the desired year from the dropdown.

No Payroll Months

If the selected year has no payroll months to display, try using 2022.

  1. Select the desired payroll month from the list.
  2. The payroll breakdown for the selected month will be displayed in the right corner of the screen.

Leave Request Management

1

Create Leave Request

Both Employees and HR Managers can create leave requests by following these steps:

  1. Click the Create button on the Leave Request page.
  2. In the modal form, fill in the leave type, start date, and end date fields.

3. Click Confirm to submit the request.

Once submitted, the leave request will appear under the Pending status filter.

2

Delete Leave Request

HR Manager

Deleting own leave request

  1. Select the leave request(s) you want to delete.
  1. Click the Delete button.
  2. In the confirmation modal, click Delete to confirm.

Deleting other employee's leave request

This process is identical to deleting your own leave request.

Employees

Deleting own leave request

  1. Select the leave request(s) you want to delete.
  1. Click the Delete button.
  2. In the confirmation modal, click Delete to confirm.

Note

Employees cannot delete other employees' requests. Only the HR Manager can do that.

3

Approve or Deny Leave Request

Important: This feature is only available to the HR Manager.

To approve or deny leave requests, the HR Manager must do the following:

  1. Select the leave request(s) to approve or deny.
  1. Depending on the current status of the request, either the Approve or Deny button will be clickable.
  2. Click the appropriate button to update the status of the selected leave request(s).

Profile Management

1

View Profile

Both roles have access to this page. To access this feature, simply navigate to the Profile page.
